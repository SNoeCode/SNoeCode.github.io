# Daily Horoscopes

### *##How it works*

A horoscope app that displays reading from all signs, with more details about charastistics of each sign giving you a deeper infomation about each sign.

Horoscopes are fetched daily from a live source and updated using GitHub actions.

### *##Tech Stack*

* HTML
* CSS
* JavaScript

### *## Features*

* 12 zodiac signs with date and images
* Daily horoscopes
* Optional deeper understand of personality traits
* Responsive design
